🚀 Technologies
Next.js 15 (App Router)
React 19
CSS Modules for styling
Google Fonts (next/font/google) — Geologica

⚙️ Scripts
bash

npm run lint     # ESLint
npm run format   # Prettier

🛠 Environment Setup
Install dependencies:

bash
npm install
Start dev server:

bash
npm run dev
